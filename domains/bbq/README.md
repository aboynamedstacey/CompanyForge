# Competitive BBQ — AI Pitmaster Organization

An AI organizational structure for competitive BBQ. 3 capabilities (Builder,
Reviewer, Scout) plus deterministic tool verification that scale from a quick
temp question to full multi-category competition prep.

## Setup

Copy this domain into your project:
```bash
cp -r domains/bbq/.claude /your/project/.claude
cp domains/bbq/CLAUDE.md /your/project/CLAUDE.md
```

## Prerequisites

- Claude Code CLI with plugins: superpowers, claude-mem, firecrawl
- Optional: BEADS for multi-session competition prep tracking

## Usage

```
/bbq what temp should I pull my brisket
/bbq build me a rub recipe for spare ribs
/bbq develop a complete rib cook protocol for KCBS
/bbq full comp prep — 4 categories for Saturday's KCBS
/bbq post-cook analysis on last weekend's results
```

The system auto-detects work type, calibrates ceremony, gathers competition
context, and staffs appropriately. Override ceremony with: "treat this as large"

## What It Does

Three workflows cover the full range:

- **Recipe Development** — rubs, sauces, injections, technique protocols, meat profiles
- **Cook Planning** — single backyard cook through full multi-category competition prep (competition prep is cook planning at Large tier, not a separate workflow)
- **Post-Cook Analysis** — score breakdowns, root cause analysis, lessons learned stored to memory for next time

Each workflow scales with ceremony tier. A quick question gets a Builder answer. A full comp prep gets Scout research, parallel per-category planning, cross-category verification, adversarial review, and two human approval gates.

## What It Can't Do

This system can't taste, smell, or assess texture. It works with documented knowledge — competition results, established techniques, prior cook data, official rules — not sensory experience. Flavor recommendations are based on what's documented to score well, not direct experience. Always apply your own palate.

All food safety guidance should be verified against USDA standards. The system flags concerns but doesn't replace proper food handling knowledge.
