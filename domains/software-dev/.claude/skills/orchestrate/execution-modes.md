# Execution Modes

## Attended (default)

Pauses at every human gate, presents findings, waits.

## Unattended

"Run this, I'll check later." Proceeds autonomously on routine work.

**Proceeds automatically:** implementation, verification, review cycles (within caps), docs, knowledge extraction.

**Banks for human review:** architecture approval, review loop escalation (>3 or >30% budget), cross-model disagreements, security blocks, budget overruns.

When human returns: present all banked decisions in order with context.

## Switching

"Continue unattended" or "stop and wait" at any gate.
